## Accessibility considerations

1. Labels should be clear and concise.
2. Users should be warned if selecting a tile will cause a change in context.
3. Carbon components should be used to create the content that displays within each tile.

## Accessibility testing

Automated, manual and screen reader accessibility verification test has been performed on the tile React Carbon component. [WCAG 2.1 Level A and AA success criteria](https://www.w3.org/TR/WCAG21/) issues have been identified and the list of open accessibility violations is available in the Carbon Component GitHub repository.
