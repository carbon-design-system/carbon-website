## Get started with charts

Refer to the [contribution guidelines](https://github.com/carbon-design-system/carbon-charts/blob/master/CONTRIBUTING.md) before you start building. If you want to develop your own components, see the [Angular Storybook](https://github.com/storybookjs/storybook/tree/next/app/angular) to get started.

Visit the [component status page](https://github.com/carbon-design-system/carbon-charts#component-status) for a complete list of available components.

### Install packages

#### Using yarn:

<br />

```bash
$ yarn add @carbon/charts @carbon/charts-angular d3
```

#### If you prefer npm:

<br />

```bash
$ npm install --save @carbon/charts @carbon/charts-angular d3
```

### Build your first chart

To start, try the `BarChart` component with the example code below:

```ts path=app.module.ts
import { NgModule } from '@angular/core';
import { BrowserModule } from '@angular/platform-browser';

import { ChartsModule } from '@carbon/charts-angular';

import { AppComponent } from './app.component';

@NgModule({
  imports: [BrowserModule, ChartsModule],
  declarations: [AppComponent],
  bootstrap: [AppComponent],
})
export class AppModule {}
```

```html path=app.component.html
<n-bar-chart
  class="n-chart"
  [data]="stackedBarData"
  [options]="stackedBarOptions"
  width="100%"
  height="400"
  #stackedBarChart
></n-bar-chart>
```

```ts path=app.component.ts
import { Component } from '@angular/core';

import { defaultColors } from '@carbon/charts';

@Component({
  selector: 'my-app',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css'],
})
export class AppComponent {
  stackedBarData = {
    // ...see next section
  };

  stackedBarOptions = {
    // ...see next section
  };
}
```

### Start the server

<br />

```bash
cd packages/angular
yarn run storybook
```

Launch your browser and go to `http://localhost:9006/`.
