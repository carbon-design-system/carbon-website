## Get started with charts

Refer to the [contribution guidelines](https://github.com/carbon-design-system/carbon-charts/blob/master/CONTRIBUTING.md) before you start building. If you want to develop your own components, see the [React Storybook](https://github.com/storybookjs/storybook/tree/next/app/react) to get started.

Visit the [component status page](https://github.com/carbon-design-system/carbon-charts#component-status) for a complete list of available components.

### Install packages

#### Using yarn:

<br />

```bash
$ yarn add @carbon/charts @carbon/charts-react d3
```

#### If you prefer npm:

<br />

```bash
$ npm install --save @carbon/charts @carbon/charts-react d3
```

### Build your first chart

To start, try the `BarChart` component with the example code below:

```jsx
import React from 'react';

import { defaultColors } from '@carbon/charts';
import { BarChart } from '@carbon/charts-react';
import '@carbon/charts/style.css';

const stackedBarOptions = {
  // ...see next section
};

class App extends React.Component {
  state = {
    stackedBarData: {
      // ...see next section
    },
  };

  render() {
    return (
      <div className="App">
        <BarChart
          data={this.state.stackedBarData}
          options={stackedBarOptions}
          width="100%"
          height={400}
        />
      </div>
    );
  }
}

// ...
```

### Start the server

<br />

```bash
cd packages/react
yarn run storybook
```

Launch your browser and go to `http://localhost:9005/`.
