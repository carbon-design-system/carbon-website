## Get started with charts

Refer to the [contribution guidelines](https://github.com/carbon-design-system/carbon-charts/blob/master/CONTRIBUTING.md) before you start building. If you want to develop your own components, see the [Vue Storybook](https://github.com/storybookjs/storybook/tree/next/app/vue) to get started.

Visit the [component status page](https://github.com/carbon-design-system/carbon-charts#component-status) for a complete list of available components.

### Install packages

#### Using yarn:

<br />

```bash
$ yarn add @carbon/charts @carbon/charts-vue d3
```

#### If you prefer npm:

<br />

```bash
$ npm install --save @carbon/charts @carbon/charts-vue d3
```

### Build your first chart

To start, try the `BarChart` component with the example code below:

```html path=hello-world.vue
<script>
  import Vue from 'vue';
  import '@carbon/charts/style.css';
  import chartsVue from '@carbon/charts-vue';

  Vue.use(chartsVue);

  export default {
    name: 'HelloWorld',
    components: {},
    data() {
      return {
        stackedBarData: {
          // ...see next section
        },
        stackedBarOptions: {
          // ...see next section
        },
      };
    },
    template:
      '<ccv-bar-chart :data="stackedBarData" :options="stackedBarOptions"></ccv-bar-chart>',
  };
</script>
```

### Start the server

<br />

```bash
cd packages/vue
yarn run storybook
```

Open browser to `http://localhost:6006/`.
