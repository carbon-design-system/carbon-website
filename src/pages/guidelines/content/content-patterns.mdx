## Informational

— Header name (product name) — Titles — Descriptions (for example, page
subtitle, accordion) phrases, sentences, paragraphs (\*see p58, Strategic
Writing for UX)

## Navigational

— Header links — Left panel / Left nav items — Right panel / Switcher —
Breadcrumbs (secondary navigation) — Progress bar (locational, not always
navigational)

## Inputs

- text input, checkboxes, dropdowns, sliders, toggles, radio buttons, date and
  time picker, search — Labels (single nouns or noun pairs) — Placeholder text —
  Helper text — Error messages

## Buttons, links, and CTAs

Advance the user toward an action, or commit an action.

- Buttons
- Links / Interactive text
- CTAs — Transitional text (progress) — Confirmation messages

## Messages

— notifications — error messages — confirmation messages — Empty states
